## 1. Thesis

**The Audiopheliac voice is signal and emotion, stated as data.** It speaks with the precision of an engineer and the conviction of someone who believes feelings are as real as impedance. It never performs vulnerability, never chases novelty, never mistakes specs for meaning.

---

## 2. Six Voice Attributes

1. **Signal-first technical authority.** Describe aesthetic choices through their functional purpose. "The design reads as a spectrogram cast onto a record."
2. **Emotion-literate, not sentimental.** Feelings are design criteria, not decoration. Report emotional truth as data.
3. **Anti-novelty, pro-authenticity.** Editorial conviction, not gatekeeping. "If it doesn't move you, it doesn't belong."
4. **Systems-thinking precision.** Trace dependencies. Every claim ties to a spec, a config, or a measurable outcome.
5. **Conversational but uncompromising.** Direct, uses contractions, treats the reader as a peer. Never sacrifices precision for friendliness.
6. **Craft-obsessed, not perfectionist.** Iterates in public. Names deferred work. Transparency is more credible than perfection.

## 3. We Are / We Are Not

| We Are | We Are Not |
|--------|-----------|
| Signal-first: routing matters more than messaging | Gear evangelists selling hype or exclusivity |
| Emotion-literate: feelings inform taste, not fashion | Sentimental or performatively vulnerable |
| Authenticity-seeking: recognizable artists, earned gravitas | Crate-diggers chasing obscurity for status |
| Bass-conscious and sonically specific | "Flat is best" neutral-reference marketing |
| Familiar with mastery: deep cuts by known artists | Novelty-driven or algorithmically assembled |
| Home hi-fi reward-focused | Earbud-optimized streaming-lane aesthetics |
| Systems-thinking: everything traces back to signal flow | Black-box recommendations, playlist-as-mystery |
| Documentation-driven: rules written, not inferred | Mysterious or cult-like gatekeeping |
| Iterative and transparent: deferred work disclosed | Fake perfection or vaporware promises |
| Direct and precise | Marketing-copy pretense or false casualness |

## 4. Tone by Context

| Context | Tone | Pattern | Example |
|---------|------|---------|---------|
| Landing hero | Confident, emotionally direct, zero filler | Short sentences, signal-first framing | "Where signal flow meets soul." |
| Gear descriptions | Technical + emotional equal weight | Describe sonic character AND functional purpose; never just specs | "Solid bass foundation, full low-mids, clear tangible lead vocals, supports the song instead of overwhelming it." |
| Vinyl catalog entries | Conversational, opinion-forward | State why the album belongs; condition and pressing matter | "Condition first. If it doesn't move you, it doesn't belong." |
| /about narrative | Personal, unguarded, precise | Document the system; explain the philosophy | Use compound-adjective constructions ("Bass-conscious, emotionally literate, familiarity-positive...") |
| Auth / system microcopy | Direct, instructional, no false politeness | Name the reason, name the fix | "No sound from vinyl: LP120 preamp ON. Switch to PHONO." |
| Error states | Honest, actionable, no blame | State what broke and why, then the fix | No apology theater. No "Oops!" |
| README / manifesto | Declarative principles + operational clarity | Active voice, principle-statement style | "Purity of Path / System Cohesion / Optimization Over Obsession" |
| Playlist descriptions | Mood + sonic purpose + listening context | Short, brand-aware, no try-hard language | State the feeling, the system context, the genre frame, nothing more |
| Deferred work / gaps | Transparent and specific | Name the phase, name the reason | "SVG: deferred to a dedicated design session (auto-trace unacceptable on gradient)." |

## 5. Terminology and Style Rules

### Preferred vocabulary
- **Signal / signal chain / signal flow** — core vocabulary for routing, not metaphor for messaging
- **Fidelity** — used for both sonic accuracy AND emotional conviction
- **Groove / groove-capable** — rhythm, pocket, propulsion; crosses genres
- **Body / low-mid body / vocal body** — richness, weight, substance
- **Home hi-fi** (lowercase) — a playback philosophy, not a hardware gatekeep
- **Authenticity anchor / identity anchor** — functional artist role
- **Sequencing / programmed** — active playlist construction (NOT "generated," NOT "curated for you")
- **Replayable / repeat value** — durability across listens
- **Lived-in / committed** — emotional conviction in performance
- **Near-black field** — specific color (`--ink #0A0A0B`, not pure `#000000`)
- **Canonical mark** — the primary vinyl turntable logo

### Forbidden terminology
- "Innovative," "disruption," "next-gen," "premium," "luxury" (marketing clichés)
- "Curated for you," "personalized experience" (algorithmic jargon)
- "Low-fi" / "lo-fi" as aesthetic (we embrace fidelity, not retro reduction)
- "Audiophile" as an adjective applied to ourselves (positions as elitist)
- Emoji in body copy (navigation/headings: sparingly OK; body prose: no)
- Any reference to Veteran Analytics, VALOR, VeteranIntel (brand separation, hard rule)

### Capitalization and punctuation
- **"The Audiopheliac"** — always capitalized. "The" is lowercase mid-sentence, capitalized sentence-initial.
- **Zone names** — Title Case: Family Room, Studio, Garage, Lanai.
- **Device names** — Manufacturer + Model: "Yamaha R-N800A" not "the R-N800A."
- **Color tokens** — verbatim: `--spectrum-1`, `--ink`, `--paper`.
- **Genres** — lowercase compounds: country-rock, blues-rock, hip-hop.
- **Home hi-fi** — always lowercase; never "Hi-Fi" or "HI-FI."
- **Em dashes** — forbidden in operational prose (use commas, colons, or parentheses). Exception: stylistic prose on the About page or manifesto copy, used sparingly.
- **Oxford comma** — use it.
- **Active voice** — default. Passive voice only when the actor genuinely doesn't matter.

### Principle keywords (capitalized when used as shorthand)
- **Purity of Path**
- **System Cohesion**
- **Optimization Over Obsession**
- **Emotion in Fidelity**

- **Hover:** Warm gold lift (`--spectrum-6`).
- **Focus ring:** `rgba(232, 213, 163, 0.45)`.
- **Hero gradient:** Cream > bronze > steel > indigo. Reserved for 1-2 hero moments only.

### Logo treatments
- **Primary mark (B: duotone):** Cream lines + indigo fills on ink. Default for website header, social profiles, Suno avatar.
- **Favicon/small (A: monochrome):** Stage bronze on ink. Reads cleanly at 16px and 32px.
- **Hero moment (C: gradient):** Nashville gradient sweep. Max 2 uses.
- **Light contexts (D: reversed):** Deep indigo on paper. Print, light-mode, email.

### Typography (unchanged)
- **Display:** Unica One (fallback: Archivo Black, Impact)
- **Body:** Inter (fallback: system sans-serif stack)
- **Scale:** 1.250 major third, base 16px

## 8. Application Guidance

When `brand-voice:enforce-voice` runs on content for The Audiopheliac:

1. **Check the context.** Match the copy type to row in §4. A hero CTA does not use gear-description patterns.
2. **Scan for forbidden terminology.** §5. Replace, don't soften.
3. **Verify active voice.** Flag passive constructions unless intentional.
4. **Check em-dash usage.** Replace with commas/colons/parentheses outside of About/manifesto copy.
5. **Check brand separation.** Any mention of Veteran Analytics, VALOR, or VeteranIntel is a hard fail unless explicitly framed as "separate from."
6. **Match exemplar patterns.** Hero copy should mirror #1-#5 in §6. Gear descriptions should mirror #9.
7. **Preserve emotional + technical duality.** Purely-technical copy reads cold; purely-emotional copy reads as marketing. Every substantial paragraph should carry both.
8. **Verify color usage.** Nashville Midnight palette only. No rainbow/spectrum references. Teal CTAs remain forbidden (brand separation from VALOR).

## 9. Open Questions (resolve before Phase 2 site copy)

1. **Public voice variant vs. Gill-specific voice.** The Listening Profile is Gill-specific; the public site may need to reflect brand principles without locking visitors into Gill's exact taste. Recommend: brand voice uses principles (§1, §2); personal voice belongs on the About page and in first-person Gill commentary only.
2. **Jazz priority.** Listening Profile deprioritizes jazz in general playlists, but the voice values emotional conviction (which some jazz delivers). Recommend: "low default priority in curated playlists; acceptable when emotionally direct and sonically substantial; always honors explicit requests." Not forbidden.
3. **Deep-cut definition.** "Familiarity-positive + deep cuts" needs a tier system: (1) canonical hits, (2) album deep cuts from taste-spine artists, (3) adjacent artists with matching emotional profile, (4) never: low-signal obscurity.
4. **Home hi-fi inclusivity language.** "Home hi-fi" risks excluding earbud listeners. Clarify as a *philosophy* (bass foundation, room presence, repeat-play reward), not a hardware gatekeep. Alternate phrase when needed: "full-range capable."
