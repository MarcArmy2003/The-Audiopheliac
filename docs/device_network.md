## 🔑 Legend
- ➡️ = wired HDMI / optical / RCA output  
- ⬇️ = signal continuation (downstream)  
- ├── / └── = split path  
- 📡 / ~~~ = wireless (Bluetooth, casting, Wi-Fi)  
- 🔊 = speaker output  
- 🎵 = turntable / audio source  
- 🎮 = gaming console  
- 🖥️ = NAS HDMI out  
- 🎛️ = hub/receiver device  
- 📺 = display
