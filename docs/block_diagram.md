## 🔑 Legend

- ├── / └── = split path  
- ≈≈≈ = wireless (Bluetooth, Casting, Wi-Fi)  
- 🎵 = turntable / audio source  
- 🎚️ = preamp / receiver  
- 🎛️ = hub/processor  
- 🎮 = gaming console  
- 📡 = wireless or network gear  
- 📺 = display (TV/monitor)  
- 🔊 = speakers / subwoofers  
- 🗄️ = NAS / storage  
- 💻 = PC workstation  
- 💼 = Work laptop  
- 🎸 = guitar equipment  
- 🖥️ = NAS HDMI out
